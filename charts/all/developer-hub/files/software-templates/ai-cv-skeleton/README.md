## First deploy

ApplicationSet **`user-neuroface-apps`** (hub `vp-gitops`) discovers GitLab repos matching `^neuroface-` in group `ws-workshop`, reads `k8s/target-cluster.yaml`, and creates an Argo CD Application targeting spoke `east` or `west`.

Verify:

```bash
oc get application -n vp-gitops neuroface-<owner>
oc get pods -n neuroface-<owner> --context=<spoke>
```
