# Support policy

This repository is a **sandbox tier** Validated Pattern.

## Support level

- Community best-effort support via [GitHub Issues](https://github.com/maximilianoPizarro/ia-computer-vision/issues)
- No Red Hat product support entitlement is included with this pattern
- Not recommended for production without your own operational review and hardening

## Getting help

1. Review the [documentation](https://maximilianopizarro.github.io/ia-computer-vision/)
2. Search existing GitHub issues
3. Open a new issue with cluster version, `clusterGroupName`, and relevant Argo CD application sync errors

Contributions and feedback are welcome. See the [Validated Patterns contributor guide](https://validatedpatterns.io/contribute/).
